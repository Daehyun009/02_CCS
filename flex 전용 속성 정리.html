[flex-container 전용 속성] display : flex; -> 내부 요소를 유연하게 배치 (감싸는 요소는 display : block 형식을 띄고 있음) flex-direction : 중심축 방향 지정 justify-content : 중심축 방향 item 정렬 align-item : 교차축 방향 item 정렬 flex-wrap : item을 한 줄 / 여러 줄로 배치 flow-flow : flex-direction + flex-wrap 한 번에 설정 align-content : wrap 설정이 된 item 묶음 교차축 방향 정렬 [item 전용 속성] order : item 순서 지정 flex-grow : item 팽창 비율 지정 (기본 값 : 0) flex-shrink : item 수축 비율 지정 (기본 값 : 1) flex-basis : item의 중심축 방향 기본 점유율(크기) flex : flex-grow + flex-shrink + flex-basis 한 번에 설정 align-self : item 별로 교차축 방향 정렬 게임 https://flexboxfroggy.com/#ko http://www.flexboxdefense.com/
